Commit messages should always include a prefix of the Jira ticket for the related work. If the Jira ticket is not obvious, ask for the ticket. The prefix should be in the format of [PROJECT-XXX]. For example, if the Jira ticket is CBDEVOPS-1234, the prefix should be [CBDEVOPS-1234].

When creating a commit, please only add files that have been changed by you, not any other changes that may be present in the repository.
